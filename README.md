# IPR_Constructora

Landing page (modelo inspirado en `rh.design`) para IPR Constructora.

## Desarrollo

La app principal está en `Bosquejo_1/ipr-next`.

```bash
cd Bosquejo_1/ipr-next
npm install
npm run dev
```

Nota: `npx serve Bosquejo_1` muestra un prototipo estático (no es la app de Next/Vercel), por eso puede verse distinto.

## Deploy (Vercel)

Se despliega desde `Bosquejo_1/ipr-next` (Next.js).
