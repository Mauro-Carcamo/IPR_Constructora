# Demo — Logo motion (3 piezas → logo completo)

Abrir `index.html` desde un servidor local para evitar bloqueos del navegador con `file://` (módulos ES + canvas).

## Opción A (recomendada)

```bash
npx serve Bosquejo_1
```

Luego abre `http://localhost:3000/logo_motion/`.

## Opción B

```bash
cd Bosquejo_1/logo_motion
python -m http.server 3000
```

Luego abre `http://localhost:3000/`.
